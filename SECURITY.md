# Security Policy

## Data Usage Policy

This application contains educational kanji data intended for personal learning use within the application interface.

### Acceptable Use
- ✅ Using the application for personal Japanese language learning
- ✅ Viewing and studying kanji through the provided interface
- ✅ Generating practice worksheets for personal use

### Prohibited Activities
- ❌ Automated scraping or mass downloading of JSON data files
- ❌ Circumventing application logic to extract bulk data
- ❌ Redistributing kanji datasets outside this application
- ❌ Using data for commercial purposes without permission
- ❌ Training machine learning models on this dataset

### Reporting Violations

If you discover unauthorized data usage or security vulnerabilities, please:

1. **DO NOT** publicly disclose the issue
2. Email the maintainers with details
3. Allow reasonable time for response and remediation

### Rate Limiting

Excessive requests to data endpoints may result in:
- IP address blocking
- Account suspension (if authentication is added)
- Legal action for systematic data theft

## Responsible Disclosure

We appreciate security researchers who:
- Report issues privately before public disclosure
- Provide detailed reproduction steps
- Allow time for fixes before publication
- Follow responsible disclosure practices

Thank you for respecting the educational mission of this project.
